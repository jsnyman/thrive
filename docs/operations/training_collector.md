# Collector Training Guide

Use this guide to train staff who register people and record intake.

## Role Summary

Collectors are responsible for:

- finding existing people
- registering new people
- recording accepted materials and weights
- confirming points awarded by the intake workflow
- requesting adjustments when a correction is needed

Collectors do not:

- record sales
- move stock between storage and shop
- apply points adjustments
- apply inventory adjustments
- view manager reports or sync conflicts

## Device and Login

- Use only the collector phone.
- Log in with a collector account.
- Confirm the app opens the people and intake workflows.
- If signal is available before opening, run `Sync Now`.

## Core Workflow

### 1. Find or Register the Person

- Search for the person first to avoid duplicates.
- If the person does not exist, create a new person with name and surname.
- Add optional details only when needed.
- Do not read back or display ID numbers or phone numbers during the interaction.

### 2. Record Intake

- Select the correct person.
- Confirm the location text before saving.
- Add each accepted material line with the correct material type and weight.
- Check the points preview before submitting.
- Save the intake only once.

### 3. Confirm Outcome

- Tell the person the points awarded.
- Explain that same-day spending depends on sync if the shop uses a separate phone.
- If the person wants to spend immediately, tell them the shop can only use balances visible after successful sync.

## Practice Scenarios

Run each scenario during training:

1. Register a new person and record a single-line intake.
2. Search for an existing person and record a multi-line intake.
3. Explain the same-day points rule to a customer.
4. Describe what to do if the app cannot sync after the intake is saved.

## Common Mistakes to Prevent

- creating a duplicate person without searching first
- entering the wrong material type
- entering the wrong weight unit or decimal
- promising immediate spending before sync has happened
- trying to use the collector phone for shop work

## If Something Goes Wrong

- If the wrong intake has not been submitted yet, correct it before saving.
- If the wrong intake was already submitted, do not try to overwrite history.
- Ask a manager to handle the correction through the proper adjustment flow.
- If sync fails, keep working offline and do not clear browser data.

## Competency Check

The collector passes training when they can:

- register a person without creating an obvious duplicate
- record intake with accurate lines and points preview review
- explain the same-day points rule clearly
- escalate mistakes without trying to bypass the audit trail
